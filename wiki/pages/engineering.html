Engineering Overview

One platform, three coordinated layers of control.

The engineering goal of Metlock is not only to recover zinc, but to do so in a way that stays tunable, measurable, and extensible. The project is organized so that each module solves a different engineering problem: controlled zinc import, predictable gene-expression calibration, and zinc binding through dedicated protein systems.

That structure matters because each module reduces uncertainty for the others. Calibration helps define safe expression windows, import work tests functional uptake under induction, and binding work expands the platform beyond one intracellular strategy.

Platform logic
Capture, import, calibrate
The system is being engineered as a pipeline rather than a single construct, so results stay comparable as the project grows.
Engineering priority
Control without overload
A central design question is how to strengthen zinc-handling performance while still protecting cell health and interpretability.
Design-Build-Test-Learn Framework

The engineering cycle guiding the project.

Design

Define controllable zinc handling

Metlock begins by identifying which parts of zinc recovery should be made tunable, measurable, and modular before moving into more specialized constructs.

Build

Assemble shared control logic

The project reuses core control architecture where possible so module outputs can be interpreted against a common experimental backbone.

Test

Compare performance with controls

Assays are organized around growth, viability, zinc depletion, expression range, and binding behavior so results remain comparable across modules.

Learn

Refine design windows

Each round is meant to narrow useful operating ranges, improve construct choices, and make future design decisions less dependent on guesswork.

Key Technical Decisions

Why the architecture is organized this way.

Shared induction language

Reusing AHL-responsive control across modules creates a more consistent basis for comparison and future redesign.

Calibration before function

Measuring sfGFP response first reduces uncertainty before the system switches into metal-handling proteins with stronger biological consequences.

Controls built into interpretation

Growth, viability, depletion, retention, and binding controls are treated as part of engineering logic, not just supporting data.
