Modular biomining platform

Metlock turns zinc recovery into a programmable biological workflow.

Metlock is an iGEM project that explores how engineered bacteria could help capture zinc more selectively and more sustainably than conventional recovery methods alone.

Instead of treating biomining as one isolated experiment, we built Metlock as a modular platform: one module controls zinc uptake, one calibrates gene expression predictively, and one is designed for external capture with on-demand release.

Why it matters

Recovering valuable metals is essential, but current methods can be energy intensive, chemically harsh, and difficult to tune with precision.

  • Targets zinc as a high-value proof of concept for biomining
  • Connects wet-lab engineering with predictive modeling
  • Builds toward reusable tools, not just a single assay result
Plain-English Overview

What Metlock is, in simple terms.

Metlock is a synthetic biology platform designed to help cells interact with zinc in more controlled ways. One part of the system helps bring zinc into the cell, another helps us predict and tune how strongly the system turns on, and a third is designed to capture zinc outside the cell and release it when needed.

Together, these modules frame Metlock as a flexible biomining toolkit rather than a one-off construct. The platform is meant to support future redesign, comparison, and extension to other metals or deployment settings.

Three Modules

Module 1

Tunable Zinc Uptake

Module 1 focuses on bringing zinc into the cell using an AHL-inducible LuxR / pLux system that drives ZnuABC. In plain English: this module is about making zinc uptake adjustable instead of always on.

  • Shared LuxR / pLux control backbone
  • ZnuABC as the uptake machinery
  • Built to study controllable zinc import
Open Module 1
Module 2

Predictive Gene Expression Calibration

Module 2 uses the same AHL-LuxR-pLux backbone with sfGFP to help predict how strongly the system turns on. It supports a simulation and database concept so design choices can be guided by calibration, not guesswork.

  • sfGFP-based calibration concept
  • Simulation-guided expression tuning
  • Foundation for reusable design rules
Open Module 2
Module 3

External Zinc Capture + On-Demand Release

Module 3 extends the platform outside the cell using an Ag43 surface-display scaffold with zinc-binding peptides. This module explores capture at the cell surface and controlled release when recovery is needed.

  • External capture strategy
  • Ag43 display scaffold
  • Designed for bind-and-release workflows
