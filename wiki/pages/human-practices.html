Human Practices

Connecting Metlock to real stakeholders and real constraints.

Human Practices is where Metlock is tested against the world beyond the bench. The project is not only about whether engineered cells can interact with zinc, but also whether that work fits real environmental needs, industrial workflows, safety expectations, and community concerns.

This page organizes the social, environmental, and stakeholder-facing side of the project into one clear place so the design can stay grounded in practical use, responsible development, and long-term implementation thinking.

Human Practices Overview

Why this project needs more than a lab-only story.

Zinc recovery sits at the intersection of resource demand, environmental damage, industrial practicality, and public trust. That means Metlock has to be evaluated not only for scientific promise, but also for where it might fit, what risks it could create, and which users or communities would actually benefit from it.

Human Practices gives the project a framework for asking those questions early. It helps the team connect laboratory design choices to stakeholder priorities, implementation realities, and responsible deployment constraints.

Community / Industry / Environmental Context

The conditions that make the problem worth solving.

The demand for critical minerals continues to grow, but conventional extraction and waste handling can impose major ecological and social costs. Zinc recovery and metal-rich wastewater contexts therefore create an important test case for whether synthetic biology can add selectivity or flexibility without simply moving the burden elsewhere.

For Metlock, that means thinking about dilute zinc streams, industrial polishing steps, contaminated environments, and what a realistic interface between biology and existing recovery infrastructure might look like.

Context callout Human Practices helps keep the project tied to real deployment environments rather than drifting into a purely theoretical construct exercise.

How Feedback Shapes Design

Turning outside input into design decisions.

Feedback on realistic use cases

Conversations with mining, environmental, and academic stakeholders can narrow which zinc concentrations, workflows, and application settings deserve priority.

Feedback on comparability and scale

External input can push the project toward shared assays, stronger controls, and clearer performance metrics that matter beyond one successful construct.

Feedback on implementation barriers

Questions around containment, handling, regulation, and industrial integration help define what the platform would need before future scale-up could be taken seriously.
