Metlock Module 3

External Zinc Capture + On-Demand Release

Module 3 extends Metlock beyond import into the cell. In plain English, this module explores whether zinc can first be captured outside the cell and then released in a controlled way when recovery is needed.

The concept centers on an Ag43 surface-display scaffold paired with zinc-binding peptides, creating a front-facing capture layer that broadens how the platform could be used in future biomining workflows.

Description

Why surface capture expands the platform.

If Module 1 is about transport into the cell, Module 3 asks whether zinc can be managed at the surface first. That matters because external capture may offer different practical advantages for handling, recovery, and downstream processing.

At a more technical level, this module explores how an Ag43 display scaffold and zinc-binding peptides could enable a bind-and-release strategy that complements the rest of the Metlock architecture.
