Metlock Module 3

Metallothionein and Display-Based Zinc Binding

Module 3 is the binding layer of Metlock. In plain English, this module asks how zinc can be captured by dedicated binding proteins, either outside the cell or inside it, so recovery can be compared across different strategies.

The module focuses on zinc-binding metallothioneins and surface-display-style binding systems, while reusing the same broader assay logic as the rest of the platform. That makes it possible to compare extracellular and intracellular binding approaches instead of treating them as isolated concepts.

Description

How Module 3 expands Metlock beyond import alone.

Module 3 exists because zinc recovery is not only about pulling metal into the cell. A useful platform also needs a way to compare how binding proteins behave when they capture zinc outside the cell versus inside the cell, especially in metal-rich environments where different handling strategies may matter.

Technically, this module focuses on zinc-binding metallothioneins and surface-display-style binding systems. By using shared assays and shared comparison logic, Module 3 gives Metlock a way to test extracellular and intracellular binding proteins under one coordinated framework rather than as separate projects.
