Metlock Module 1

pLuxR-ZnuABC Tunable Zinc Import

Module 1 is the import engine of Metlock. In plain English, this module is about giving cells a controllable way to bring zinc inside only when the team wants that uptake to happen.

The design uses a constitutive LuxR helper together with AHL-driven pLuxR expression of ZnuA, ZnuB, and ZnuC. That shared control logic makes zinc import adjustable across induction conditions, which is important for balancing recovery performance with cell health.

Description

How zinc import is controlled on command.

Module 1 addresses a simple but important problem: zinc recovery is only useful if the cells can import zinc efficiently without being pushed into stress too quickly. Rather than leaving uptake permanently active, Metlock places import under a tunable induction system so the team can choose when and how strongly the transport machinery is expressed.

In technical terms, the module couples a constitutive LuxR helper with AHL-responsive pLuxR expression of ZnuA, ZnuB, and ZnuC. ZnuA functions as the binding component, ZnuB provides membrane transport, and ZnuC powers the system through ATP-dependent activity. Together, they create a controllable zinc import circuit that can be tested across different AHL levels and compared against growth, viability, depletion, and retention controls.
