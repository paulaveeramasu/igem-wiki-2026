Metlock Module 1

Tunable Zinc Uptake

Module 1 is the entry point for zinc into the Metlock platform. In plain English, this module is about helping cells pull in zinc in a way that can be adjusted instead of left permanently on.

More specifically, Module 1 uses an AHL-inducible LuxR / pLux control system to drive ZnuABC, the transporter responsible for zinc uptake. This makes Module 1 the core import layer that the rest of the platform can build around.

Description

What this module is designed to do.

The purpose of Module 1 is to test whether zinc uptake can be made controllable. Instead of relying on a fixed transport state, the system is designed so that zinc import responds to an external induction signal and can be studied across a range of conditions.

At the technical level, this module places ZnuABC under the LuxR / pLux backbone so the team can compare expression strength, transport behavior, and practical assay outcomes as induction changes.
